# Lychee Bass - 功能路线图

本文档记录项目的规划功能和改进方向，按优先级和实现难度分类。

## 🎯 近期目标（短期可实现）

### 指板增强
- [ ] **横向指板视图** - 提供水平布局，更接近真实演奏视角
- [ ] **五弦贝斯支持** - 增加 B-E-A-D-G 调弦选项
- [ ] **左手模式** - 镜像翻转指板，适配左手贝斯手
- [ ] **真实采样音色** - 使用 Tone.Sampler 替换合成器，加载真实贝斯采样
- [ ] **调弦切换器** - 支持标准调弦、Drop D、开放调弦等

### 节奏与练习工具
- [ ] **节拍器** 
  - BPM 可调（40-240）
  - 支持 4/4、3/4、6/8 等拍号
  - 重音设置
  - 音量控制
- [ ] **音阶练习器** - 自动播放选定音阶，用户跟弹，支持速度调节
- [ ] **录音功能** - 使用 MediaRecorder API 录制演奏片段，支持回放和下载
- [ ] **波形显示** - 实时显示录音波形，帮助分析节奏准确度

### 学习进度系统
- [ ] **LocalStorage 持久化** - 保存练习记录、统计数据
- [ ] **学习日志** - 记录每日练习内容、时长、BPM 进度
- [ ] **成就系统** - 里程碑奖励（如"首次完成 12 小节布鲁斯"）
- [ ] **练习打卡日历** - 可视化展示练习连续天数

### 音名训练增强
- [ ] **难度分级** - 初级（单弦）、中级（双弦）、高级（全指板）
- [ ] **时间挑战模式** - 限时答题，记录最快速度
- [ ] **自定义范围** - 只训练指定品位或弦

## 🚀 中期扩展

### 听力训练模块
- [ ] **音程识别** - 播放两个音，用户判断音程类型
- [ ] **和弦识别** - 播放和弦，用户在指板上点出组成音
- [ ] **旋律听写** - 播放简短旋律，用户在指板上复现

### 曲目与谱面系统
- [ ] **曲目库管理**
  - 想学的 / 在练的 / 已掌握 三个清单
  - 支持添加标签、难度评级、笔记
- [ ] **Bass Line 播放器**
  - 导入简化谱格式（如 JSON）
  - 滚动谱表 + 指板同步高亮
  - 速度调节（0.5x - 2x）
  - 循环播放指定小节
- [ ] **伴奏轨功能**
  - 内置常见 Drum Loop（摇滚、蓝调、放克）
  - 简单和弦进行伴奏（使用 Tone.js 合成）
  - 用户可上传自己的伴奏音频

### 用户内容系统
- [ ] **笔记系统**
  - 截图指板当前状态
  - 添加文字注释、标签
  - 关联到曲目或练习日志
- [ ] **练习片段管理** - 保存录音片段，添加标签和描述，方便回顾

### 和弦工具
- [ ] **和弦查询器** - 输入和弦名（如 Cmaj7），显示指板上的按法
- [ ] **和弦进行可视化** - 输入和弦进行（如 I-IV-V），自动标注根音位置

## 🌟 长期愿景

### 教程与内容平台
- [ ] **内嵌教程文章** - Markdown 编写，嵌入可交互指板演示
- [ ] **视频教程集成** - 嵌入 YouTube 视频，支持时间戳跳转
- [ ] **练习计划推荐** - 根据用户水平生成学习路径

### 社区功能
- [ ] **公共练习房间** - 使用 WebRTC 实现在线 jam session
- [ ] **分享功能** - 分享练习记录、指板配置、自定义 bass line
- [ ] **评论与反馈** - 用户可对曲目、教程留言交流

### AI 辅助
- [ ] **自动扒带** - 上传音频，AI 识别 bass line 并生成谱面（技术难度高）
- [ ] **演奏评估** - 录音后 AI 分析节奏准确度、音准偏差
- [ ] **个性化推荐** - 根据练习数据推荐下一步学习内容

## 🔧 技术优化

### 性能与体验
- [ ] **PWA 支持** - 离线可用，添加到主屏幕
- [ ] **移动端优化** - 防止滚动冲突，优化触摸体验
- [ ] **深色/浅色主题切换**
- [ ] **键盘快捷键** - 空格播放/暂停，方向键切换等

### 数据与存储
- [ ] **IndexedDB 支持** - 存储大量练习数据和录音
- [ ] **云同步（可选）** - 多设备数据同步
- [ ] **数据导出** - 导出为 JSON 或 CSV，方便备份

### 音频引擎
- [ ] **低延迟优化** - 减少点击到发声的延迟
- [ ] **采样库管理** - 支持用户上传自定义采样
- [ ] **效果器链** - 压缩、EQ、混响等基础效果

## 📋 已知问题与改进

### 当前 Bug
- [ ] 移动端拖动时可能触发浏览器滚动
- [ ] 音色较简陋，需要更真实的音色

### 代码质量

- [ ] 添加 E2E 测试
